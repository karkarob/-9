Что такое SillyTavern?

SillyTavern — это форк популярного TavernAI, предоставляющий расширенные возможности для создания и взаимодействия с AI-персонажами. Это приложение позволяет вам общаться с искусственным интеллектом через различные API, создавать уникальных персонажей и погружаться в увлекательные ролевые сценарии.

Интуитивный интерфейс
Создание персонажей
Мощные API

Основные возможности:

  • Поддержка множества AI API (OpenAI, KoboldAI, NovelAI и другие)
  • Расширенные инструменты создания персонажей
  • Групповые чаты с несколькими AI-персонажами
  • Гибкие настройки генерации текста
  • Плагины и расширения для дополнительного функционала
  • Визуально приятный и настраиваемый интерфейс

Готовы окунуться в мир магии AI? Начните с руководства по установке!

Начать путешествие

🪄 Установка SillyTavern

Правильная установка — первый шаг к магии общения с AI. Следуйте этим инструкциям, чтобы подготовить ваше цифровое заклинание.

1Подготовка ингредиентов

Перед установкой убедитесь, что у вас есть:

  • Node.js (версия 18 или выше) — скачать здесь
  • Git — скачать здесь
  • Стабильное интернет-соединение
  • Не менее 2ГБ свободного места на диске

2Клонирование репозитория

Откройте терминал (командную строку) и выполните:

git clone https://github.com/SillyTavern/SillyTavern.git

Перейдите в созданную директорию:

cd SillyTavern

3Установка зависимостей

Выполните команду:

npm install

Это загрузит все необходимые компоненты для работы SillyTavern.

4Запуск приложения

После завершения установки запустите SillyTavern:

node server.js

Если все сделано правильно, вы увидите сообщение о успешном запуске сервера.

5Открытие в браузере

Откройте ваш браузер и перейдите по адресу:

http://localhost:8000

Теперь вы должны увидеть интерфейс SillyTavern!

Совет: Если порт 8000 занят, SillyTavern автоматически попробует использовать следующий доступный порт. Проверьте терминал для точного адреса.
Важно: Не закрывайте терминал во время работы SillyTavern — это остановит сервер.

Поздравляем! Вы успешно установили SillyTavern. Теперь перейдем к настройке.

Перейти к настройке

⚙️ Настройка SillyTavern

Настройте ваше магическое зеркало общения под свои предпочтения для идеального опыта.

1Базовая конфигурация

После первого запуска выполните первоначальную настройку:

  • Выберите язык интерфейса в настройках
  • Настройте тему оформления (темная/светлая)
  • Установите предпочтительные горячие клавиши
  • Настройте параметры сохранения диалогов

2Настройки генерации текста

Temperature (Температура)

Контролирует случайность ответов. Значение 0.7-0.9 для творческих ответов, 0.3-0.6 для более предсказуемых.

Top-P (Ядерная выборка)

Ограничивает выбор слов только из наиболее вероятных. Рекомендуется 0.8-0.95.

Max Length (Макс. длина)

Максимальное количество токенов в ответе. Обычно 100-300 для баланса между детализацией и краткостью.

Frequency Penalty (Штраф за частоту)

Снижает вероятность повторяющихся фраз. Используйте 0.1-0.5 для уменьшения повторов.

3Расширенные настройки

  • Контекстное окно: Настройте сколько предыдущих сообщений AI будет помнить
  • Плагины: Активируйте нужные расширения (изображения, TTS, и т.д.)
  • Безопасность: Настройте параметры конфиденциальности и фильтры содержимого
  • Резервное копирование: Настройте автоматическое сохранение ваших данных
Профессиональный совет: Начните с preset-настроек "Storyteller" или "Roleplay", затем постепенно ajustируйте параметры под свои нужды.
Перейти к API ключам

🔑 API Ключи и подключение сервисов

Подключите магические источники силы для вашего SillyTavern. Выберите подходящий AI-сервис.

Поддерживаемые API сервисы

Сервис Тип Сложность Стоимость
OpenAI (GPT-3.5/GPT-4) Официальное API 🐉 Легко Платный
KoboldAI Локальный/Сетевой 🐉🐉 Средне Бесплатный
NovelAI Официальное API 🐉 Легко Подписка
Oobabooga Локальный 🐉🐉🐉 Сложно Бесплатный
Claude AI Официальное API 🐉 Легко Платный

1Получение API ключей

Для платных сервисов (OpenAI, NovelAI):

  1. Зарегистрируйтесь на сайте провайдера
  2. Перейдите в раздел API или Billing
  3. Сгенерируйте новый API ключ
  4. Скопируйте ключ в безопасное место
Внимание! Никогда не делитесь вашими API ключами. Они являются платными и могут быть использованы злоумышленниками.

2Настройка подключения

В SillyTavern перейдите в раздел Connections → API и:

  1. Выберите ваш провайдер из списка
  2. Вставьте API ключ в соответствующее поле
  3. Настройте дополнительные параметры (если нужно)
  4. Нажмите "Connect" и проверьте соединение
Совет: Для локальных решений (KoboldAI, Oobabooga) убедитесь, что сервер запущен и доступен по указанному адресу.

3Тестирование подключения

После настройки:

  • Создайте тестового персонажа
  • Начните простой диалог
  • Проверьте скорость и качество ответов
  • Настройте параметры генерации под ваш стиль
Создать персонажей

🎭 Создание и настройка персонажей

Дайте жизнь вашим фэнтезийным персонажам с помощью детальной настройки их личностей и характеристик.

1Создание нового персонажа

Перейдите в Characters → Create New и заполните:

  • Имя: Как будет звать вашего персонажа
  • Описание: Основные физические характеристики
  • Личность: Черты характера, манеры поведения
  • Сценарий: Контекст и окружение персонажа
  • Приветственное сообщение: Первая фраза при знакомстве

Пример: Эльфийский маг

Описание: Высокий эльф с серебристыми волосами и голубыми глазами, одет в синие магические robes.

Личность: Мудрый, терпеливый, немного загадочный, обладает чувством юмора.

Сценарий: Магическая академия в древнем лесу.

2Расширенные настройки персонажа

Используйте дополнительные поля для детализации:

  • Alternate Greetings: Альтернативные приветствия
  • Post History: Примеры предыдущих диалогов
  • World Info: Информация о мире персонажа
  • Persona: Детальное описание психологического портрета

3Импорт и экспорт персонажей

SillyTavern поддерживает форматы:

  • .json: Нативный формат SillyTavern
  • .character: Формат Character Card
  • .png: Картинки с встроенными метаданными
Совет: Используйте онлайн-базы персонажей (например, Chub.ai) для поиска и загрузки готовых персонажей.

4Групповые чаты

Создайте взаимодействие между несколькими персонажами:

  1. Создайте или импортируйте нескольких персонажей
  2. Перейдите в Group Chat → Create New
  3. Добавьте выбранных персонажей в группу
  4. Настройте очередность ответов и взаимодействия
Узнать советы

💡 Мастерские советы и хитрости

Секреты опытных магов SillyTavern для идеального опыта ролевых игр и чатов.

Советы по созданию персонажей

🐉 Детализация

Чем подробнее описание персонажа, тем consistentнее его поведение. Добавьте мелочи: привычки, предпочтения, манеру речи.

🎭 Примеры диалогов

Добавьте 5-10 примеров типичных реплик персонажа в разных ситуациях. Это поможет AI понять стиль общения.

📖 World Info

Создайте отдельные записи о ключевых местах, организациях и понятиях вашего мира для согласованности lore.

⚖️ Баланс

Не перегружайте персонажа деталями. 500-1000 токенов описания обычно достаточно для качественной работы.

Оптимизация диалогов

  • Регулярно чистите контекст: Длинные диалоги могут замедлять работу и снижать качество ответов
  • Используйте Summary: Включайте автоматическое суммирование длинных бесед
  • Настройте глубину памяти: Experimentируйте с количеством запоминаемых сообщений
  • Исправляйте ошибки AI: Редактируйте сообщения персонажа чтобы скорректировать его поведение

Технические советы

  1. Резервное копирование: Регулярно экспортируйте своих персонажей и настройки
  2. Эксперименты: Создайте копии персонажей для тестирования разных настроек
  3. Расширения: Изучите доступные плагины для дополнительного функционала
  4. Сообщество: Присоединитесь к Discord и Reddit сообществам для обмена опытом
Важно: Помните об этике использования AI. Уважайте авторские права и не создавайте вредоносный контент.
Финальный совет: Не бойтесь experimentировать! SillyTavern предлагает безграничные возможности для творчества. Находите свой уникальный стиль взаимодействия с AI.
